# Change Log

All notable changes to the "codedemo" extension will be documented in this file.

## [0.1.2]

- 修复自动补全时没有替换模板字符的 bug

## [0.1.1]

> 新增模版字符替换规则

- 【【文件名带扩展名】】--> 当前编辑的文件名（带后缀）
- 【【fileNameWithExt】】 --> 当前编辑的文件名（带后缀）
- 【【文件名】】--> 当前编辑的文件名（不带后缀）
- 【【fileName】】 --> 当前编辑的文件名（不带后缀）

## [0.1.0]

> 初始化版本，支持以下功能

- 代码片段仓库配置
- 代码片段同步到远程仓库
- 保存代码片段
- 插入代码片段
- 删除代码片段
- 代码片段智能补全提示
